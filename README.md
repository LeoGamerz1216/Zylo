Zylo 

> **The Premium, Brand-Free Video Experience.** > Developed by **Pavan Tej** | Made by **Oasis Centers India**

Zylo is a high-performance, open-source video player built for those who want the content of YouTube without the noise. No logos, no trackers, and a completely custom "Premium" UI.

![Zylo Banner](https://via.placeholder.com/800x200?text=Zylo+Premium+Interface) ## ✨ Why Zylo?
* **Pure Content:** The original video from the database, stripped of the YouTube player and branding.
* **Privacy First:** No Google tracking, no ads, just your videos.
* **Premium Aesthetic:** Built with a modern, glassmorphic UI designed for a high-end experience.
* **Oasis Engine:** Optimized for speed and low memory usage.

## 🚀 Getting Started

### Prerequisites
* **Node.js** (LTS Version)
* **Yarn** (`npm install --global yarn`)

### Installation
If you have a slow connection, use our optimized install command:

```bash
# Clone the repository
git clone [https://github.com/YourUsername/Zylo.git](https://github.com/YourUsername/Zylo.git)

# Install dependencies with high-timeout for large icon sets
yarn install --network-timeout 1000000 --network-concurrency 1
Running in Development
Bash
yarn run dev
🛠 Project Structure
src/main: The "Brain" - Handles video extraction and system logic.

src/renderer: The "Beauty" - Vue.js components and the Zylo UI.

documentation/: Deep-dive guides on vision and setup.
